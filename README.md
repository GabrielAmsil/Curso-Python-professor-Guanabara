# Curso de Python - Curso em Vídeo

Repositório criado para armazenar os desafios resolvidos durante o curso de Python do professor Gustavo Guanabara.

## Estrutura

O curso é dividido em três módulos:

- Mundo 1 — Fundamentos do Python
- Mundo 2 — Estruturas de controle
- Mundo 3 — Estruturas compostas e funções

Cada pasta contém os desafios resolvidos em Python.

## Objetivo

Este repositório tem como objetivo:

- Praticar lógica de programação
- Aprender os fundamentos do Python
- Registrar minha evolução na linguagem

## Tecnologias utilizadas

- Python 3
- Visual Studio Code
